# FaunaPassage

Software for Faunapassages based on:
 * Sodaq Sara R410 AFF
 * 1x PIR Sensor (PIN D4)
 * 2x Breakbeam sensor (PIN D2, D3)
 * KPN Things
 * Ubitdots
 
      North  (D2)       PIR (D4)
+------------------+  
|                  |      
| FaunaPassage     |
|                  |
|                  |
+------------------+
     South   (D3)
     
Measures:
* PIR Detections
* Break beam interrupts
* Traffic from North -> South (Interval default 30s)
